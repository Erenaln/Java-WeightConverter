# Weight Converter

## Description
A simple, interactive Java console application that allows users to convert weights between pounds (lbs) and kilograms (kg). 

## Features
* **Interactive Menu:** Prompts the user to select the desired conversion type.
* **Pounds to Kilograms:** Accurately converts lbs to kg (1 lb = 0.453 kg).
* **Kilograms to Pounds:** Accurately converts kg to lbs (1 kg = 2.204 lbs).
* **Defensive Programming:** Includes basic error handling to catch invalid menu selections and guide the user appropriately.

## Prerequisites
* Java Development Kit (JDK) installed on your system.
* An IDE (like Eclipse or IntelliJ IDEA) or a standard terminal/command prompt.

## How to Run
1. Clone this repository or download the `WeightConverter.java` file.
2. Open your terminal or command prompt.
3. Navigate to the directory containing the file.
4. Compile the Java file using the following command:
   
```bash
   javac WeightConverter.java
